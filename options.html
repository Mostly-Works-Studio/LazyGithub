Quick Reference

Template tokens — use in inputs values and buildComment
{version} {profile} {prTitle} {branchName}
Workflow conditions

"if": "version:contains:VALUE" or "if": "version:notContains:VALUE". Omit if for an unconditional default rule.

excludedRepos

Repo strings or regex patterns (e.g. "myorg/.*", "-helm-values$"). No buttons are injected on matching repos.

commentAuthorFilter

GitHub usernames or regex patterns. When non-empty, deploy buttons appear only on matching authors' comments.

buildButton.hiddenOnStates

PR states that suppress the Build button (e.g. ["merged", "closed"]). Leave empty to always show.
